REPOSITORIO DE EJERCICIOS JAVA

Los distintos temas de las clases de Java vienen con un conjunto de ejercicios propuestos y sus correspondientes soluciones.
En este repositorio ire subiendo los ejercicios extra que se planteen cada curso, así como aquellos que formen parte de los
distintos examenes que vayamos realizando.

PROPOSITO: El proposito de este repositorio es ir registrando todos los ejercicios que se vayan planteando en clase. De esta forma, los alumnos que requieran más ejercicios para su práctica, los encontraran aquí. 

ESTADO
La idea es que este proyecto este en constante construcción, enriqueciéndolo con los distintos ejercicios que vayamos planteando cada curso. Empiezo en el curso 2023-24, volcando todos los ejercicios que tengo de cursos anteriores.

USO
La explicación de la teoría y algunos ejercicios resueltos están disponibles en el sitio dat-science.com
Este repositorio es un repositorio de ejercicios extra, donde se dispone de un fichero con los enunciados de dichos ejercicios y sus soluciones. El código de las soluciones se proporciona en un proyecto java implementado con el IDE: IntelliJ IDEA o en algunos casos puede que con NetBeans. El JDK empleado también puede variar ya que lo lógico es que lo vayamos actualizando cada año y por tanto, los ejercicios subidos un curso podrían tener un JDK distinto a los de otro curso.
