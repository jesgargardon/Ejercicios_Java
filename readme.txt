REPOSITORIO DE EJERCICIOS JAVA

Los distintos temas de las clases de Java vienen con un conjunto de ejercicios propuestos.
En este repositorio ire subiendo las soluciones de los ejercicios que se planteen cada curso, así como aquellos que formen parte de los distintos examenes que vayamos realizando.

PROPOSITO: El proposito de este repositorio es ir registrando todos los ejercicios que se vayan planteando en clase. De esta forma, los alumnos que requieran más ejercicios para su práctica, los encontraran aquí. 

ESTADO
La idea es que esté en constante construcción, enriqueciéndolo con los distintos ejercicios que vayamos planteando cada curso. Empiezo en el curso 2023-24, volcando todos los ejercicios que tengo de cursos anteriores.
